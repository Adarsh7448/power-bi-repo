# Power BI Workshop – Making Interactive Dashboards with Power BI

This repository contains all the materials, recordings, and assignments from the **Power BI Workshop**, focused on building a strong foundation in data preparation, modeling, visualization, and reporting using Power BI.

---

## Workshop Overview

The workshop was conducted over **two days**, covering both **core concepts** and **hands-on report building** using real-world data.

---

## What We Covered

### Day 1: Data Preparation & Modeling
- Introduction to Power BI ecosystem
- Connecting to data sources
- Data cleaning using **Power Query**
  - Removing duplicates and errors
  - Changing data types
  - Renaming columns and tables
  - Creating calculated columns where required
- Understanding **data modeling**
  - Fact vs Dimension tables
  - Relationships and cardinality
  - Star schema best practices
- Building the foundation for scalable and performant reports

### Day 2: Visualization & Reporting
- Understanding how to choose the right visual
- Building core report visuals:
  - KPI Card (Total Sales)
  - Category-wise performance analysis
  - Time-based trend analysis
  - Customer-level detailed tables
- Formatting and improving report readability
- Basic storytelling with visuals
- Best practices for report layout and usability
